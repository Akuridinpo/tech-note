# Shell 变量
## 系统变量
	例如`$PATH $HOME $SHELL`
## 用户自定义变量
+ 定义方法：变量=值
+ 撤销变量：`unset 变量`
+ 申明不能撤销的静态变量： `readonly 变量`
+ 输出变量时必须 `$变量`
+ 变量可以是指令，定义方法：``变量=`指令`或者 变量=$(指令)
+ 在 `/etc/profile` 里，可以用 `export 变量名=变量值` 将 shell 变量输出为环境变量/全局变量（配置完成后需要 `source 配置文件` 让配置生效）
