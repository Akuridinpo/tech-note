# Crond 任务调度的时间规则
+ 占位符

|         | 含义        | 范围   |
| ------- | --------- | ---- |
| 第一个 `*` | 一小时中的第几分钟 | 0-59 |
| 第二个 `*` | 一天当中第几小时  | 0-23 |
| 第三个 `*` | 一个月当中第几天  | 1-31 |
| 第四个 `*` | 一年当中第几月   | 1-12 |
| 第五个 `*` | 一周当中星期几   | 0-7  |
+ 特殊符号
  

| 特殊符号 | 含义        |
| ---- | --------- |
| *    | 代表任何时间    |
| ，    | 代表不连续时间   |
| -    | 代表连续的时间范围 |
| */n  | 代表多久执行一次  |
